# Paper ingestor — spec notes

## Summary

The most useful thing for Laura: a live updater that reliably captures papers applying or developing new biodesign tools, and feeds that information into the model database.

## What this does

Provides the ability to simply add a paper to a large database. It gets automatically categorised and clustered.
